# Amazon-Web-Scraper-Project
“Amazon web scraping project using Python, BeautifulSoup, and Pandas to collect product data for basic price tracking.”

This project is a simple Python-based web scraper that extracts product data from Amazon.

## Features
- Scrapes product title and price
- Stores data into a CSV file
- Can be used for basic price tracking

## Tools Used
- Python
- BeautifulSoup
- Requests
- Pandas

## How It Works
The script sends a request to an Amazon product page, parses the HTML, and extracts useful data like product name and price.

## Note
Amazon may block requests, so headers are used to simulate a real browser.
